# MVP Roadmap

## Phase 1: Foundation

- Monorepo, Docker, PostgreSQL, Redis.
- Auth, roles, permissions.
- Candidate profile and resume CRUD.
- Company profile and vacancy CRUD.
- Public job search and SEO job pages.

## Phase 2: Hiring Workflow

- Applications and statuses.
- Employer dashboard.
- Candidate dashboard.
- Basic notifications.
- Chat MVP with polling or WebSocket gateway.

## Phase 3: AI and Subscriptions

- Resume improvement.
- Job recommendations.
- Cover-letter generation.
- Premium and Business plan limits.
- Stripe checkout and webhook handling.

## Phase 4: Admin and Scale

- Admin moderation panel.
- AI usage and billing dashboards.
- Queue workers for AI, email, analytics.
- Search indexing optimization.
- Production monitoring and backups.

## Phase 5: SaaS Hardening

- Company teams and invitations.
- Advanced employer analytics.
- Candidate database access.
- Anti-spam and fraud controls.
- Multi-region deployment readiness.
